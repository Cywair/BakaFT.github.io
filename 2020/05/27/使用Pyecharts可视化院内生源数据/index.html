Pyecharts 简介

ECharts 是一个使用 JavaScript 实现的开源可视化库，涵盖各行业图表，满足各种需求。凭借着良好的交互性，精巧的图表设计，得到了众多开发者的认可。

而 Python 是一门富有表达力的语言，很适合用于数据处理。当数据分析遇上数据可视化时，Pyecharts 诞生了。

前言

最近在学Python，爬取到的很多数据如果可以用可视化的方式来展现的话，那肯定很酷。说到可视化最常见的就是各种图表，于是我就找到了这个东西，正好可以给院内的生源和分流情况做一个分析。
